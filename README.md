#AmiResumegen

A fast, interactive, and fully client-side web application designed to help students and professionals build beautifully formatted, ATS-friendly resumes in minutes. 

With AmiResumegen, you can focus on your content while the application handles the design, offering a live preview and instant high-quality PDF exports.

##Key Features

* ** Live Real-Time Preview:** See exactly how your resume looks as you type. No refreshing required.
* ** Dynamic Theming:** Choose from pre-built professional templates or customize your own brand colors and fonts using the native color picker.
* ** Flexible Sections:** Dynamically add or remove education, projects, and work experience fields with a single click.
* ** Profile Picture Support:** Upload and instantly crop a profile photo directly into your resume header.
* ** High-Resolution PDF Export:** Built-in integration with `html2pdf.js` to ensure your final downloaded resume is crisp and print-ready.

## Technology Stack

This project is built using pure, vanilla web technologies to ensure maximum performance and zero server costs.

* **HTML5:** Semantic structure and dynamic input handling.
* **CSS3:** Custom variables for theming, responsive flexbox layouts, and print-specific styling.
* **JavaScript (Vanilla):** DOM manipulation, state management, and real-time rendering.
* **html2pdf.js:** Client-side HTML-to-PDF generation.
* **FontAwesome:** Professional iconography.

##  How to Run Locally

Because this project is entirely client-side, you don't need Node.js or a backend server to run it.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/tamitkumar24/AmiResumegen.git](https://github.com/tamitkumar24/AmiResumegen.git)
    ```
2.  **Open the project folder:**
    Navigate to the directory where you cloned the files.
3.  **Launch the app:**
    Simply double-click the `index.html` file to open it in your default web browser. 

*(Optional: For the best development experience, you can use the "Live Server" extension in VS Code).*
