About This Project

AmiResumegen is a client-side web application designed to help students and professionals create beautifully formatted, ATS-friendly resumes in minutes. With live preview rendering and instant PDF downloads, it entirely removes the hassle of formatting documents manually.

Technology Stack

HTML5 CSS3 JavaScript html2pdf.js
